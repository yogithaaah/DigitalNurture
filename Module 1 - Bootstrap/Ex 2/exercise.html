📁 Bootstrap Folder Structure & JavaScript Plugins

Exercise 2.1 – Bootstrap Folder Overview

bootstrap/
├── css/
│ ├── bootstrap.min.css
│ ├── bootstrap.css
│ └── bootstrap.min.css.map
├── js/
│ ├── bootstrap.bundle.min.js
│ ├── bootstrap.min.js
│ └── bootstrap.bundle.min.js.map
└── icons/ (optional)

The css/ folder contains the compiled Bootstrap stylesheets.
The js/ folder includes essential JavaScript plugins and dependencies.
If downloaded, icons/ provides Bootstrap’s SVG icons.

Exercise 2.2 – Including Bootstrap JavaScript

To include Bootstrap’s JavaScript plugins, use the following script tag:

<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>

The bootstrap.bundle.min.js includes both Bootstrap's JavaScript and Popper.js, enabling components like dropdowns and tooltips.
